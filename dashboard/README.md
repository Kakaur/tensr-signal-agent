# Dashboard

React + Vite frontend for Tensr Signal Agent.

## Scripts

- `npm run dev`: Start frontend dev server.
- `npm run build`: Build production assets.
- `npm run preview`: Preview production build.
- `npm run lint`: Run ESLint.
- `npm run start`: Start backend (`uvicorn`) and frontend together.

## Backend Contract

The dashboard expects the FastAPI backend to run at:

- `http://localhost:8000`
